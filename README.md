# 🚗 DriveSafe Eyes

ระบบตรวจจับอาการง่วงนอนและหาวของผู้ขับขี่แบบเรียลไทม์  
ใช้กล้องเว็บแคมในการวิเคราะห์ใบหน้าและดวงตา ด้วยเทคโนโลยี **MediaPipe FaceMesh**  
เพื่อช่วยเตือนผู้ขับเมื่อมีแนวโน้ม "ง่วงหลับใน" หรือ "หาวหลายครั้งในช่วงสั้น ๆ"

---

## 🧠 คุณสมบัติเด่น (Features)

- 👁️ ตรวจจับการ **หลับตาติดต่อกัน** ด้วยค่า EAR (Eye Aspect Ratio)
- 😪 ตรวจจับการ **หาว** ด้วยค่า MAR (Mouth Aspect Ratio)
- 🔁 ระบบ **กรองสัญญาณ (Smoothing & Majority Vote)** เพื่อลดการแจ้งเตือนผิดพลาด
- 🔊 เล่นเสียงเตือน 2 ระดับ (Alert1, Alert2) และเสียงเตือนพิเศษเมื่อหาวหลายครั้ง
- 🧾 บันทึกเหตุการณ์ทุกครั้งลงใน `logs/events.csv`
- ⚙️ ตั้งค่าพารามิเตอร์ได้ในไฟล์ `config.yaml`
- 🧩 รองรับ **การ Calibrate อัตโนมัติ** เพื่อหาค่า EAR Threshold เฉพาะบุคคล
- 🖥️ อินเทอร์เฟซแบบ Full Screen พร้อมโลโก้ DriveSafe Eyes
- ✅ ป้องกัน Error จากกล้องและเสียง (try/except ครอบคลุม)

---

## ⚙️ การตั้งค่าใน `config.yaml`

| พารามิเตอร์ | ค่าเริ่มต้น | คำอธิบาย |
|--------------|--------------|-----------|
| `ear_closed_threshold` | 0.25 | ค่าขีดจำกัดของ EAR ต่ำกว่านี้ถือว่าตาปิด |
| `closed_min_seconds` | 1.2 | เวลาที่ต้องปิดตาต่อเนื่องก่อนแจ้งเตือน |
| `alert2_after_more_seconds` | 2.0 | เวลาที่เพิ่มจาก Alert1 เพื่อแจ้งเตือนระดับ 2 |
| `smoothing_alpha` | 0.4 | ค่าความเนียนของค่า EAR (0.3–0.6 แนะนำ) |
| `majority_frames` | 3 | จำนวนเฟรมที่ใช้โหวตค่า “ตาปิด” |
| `min_face_ratio` | 0.05 | อัตราขนาดหน้าต่ำสุด (เล็กกว่านี้ถือว่าไกลเกินไป) |
| `min_brightness` | 30 | ความสว่างขั้นต่ำ (ภาพมืดจะไม่ตรวจจับ) |
| `allow_alert_on_low_quality` | true | อนุญาตแจ้งเตือนแม้ภาพไม่ชัด |
| `camera_index` | 0 | หมายเลขกล้อง (0 = กล้องหลักในเครื่อง) |
| `fps_target` | 30 | อัตราเฟรมเป้าหมาย (เพิ่มเป็น 60 ถ้ากล้องรองรับ) |
| `yawn_notice_min_show_sec` | 5.0 | ระยะเวลาที่ป้าย “Yawning Detected” ค้างบนจอ |
| `yawn_notice_cooldown_sec` | 5.0 | เวลาพักก่อนอนุญาตให้แจ้งเตือนหาวซ้ำ |
| `yawn_series_needed` | 3 | จำนวนครั้งของการหาวที่จะถือว่า “หาวหลายครั้ง” |
| `yawn_series_window_sec` | 60 | ช่วงเวลาที่ใช้ในการนับจำนวนหาว |
| `ui_scale` | 1.4 | ขนาด UI โดยรวม (1.0 = ปกติ, 1.2–1.6 = ใหญ่ขึ้น) |



## ⚙️ วิธีติดตั้ง 
## หมายเหตุ ใช้ Python 3.9 ขึ้นไป และห้ามใช้ python ที่สูงกว่า Python 3.10 เนื่องจากบาง library ยังไม่รองรับกับ Python รุ่นใหม่กว่า 3.10  (แนะนำ python 3.10.x)

1. สร้าง virtual environment:
```bash
   python -m venv .venv
   ```
   ```bash
   .venv\Scripts\activate
   ```
2. ติดตั้ง dependencies:
```bash
   pip install -r requirements.txt
   ```

## 📕 วิธีการใช้งาน

## 🧪 เมื่อใช้งานครั้งแรก ต้องคาลิเบรทก่อน (Calibrate EAR) เพื่อสอนระบบให้รู้จักตาของคุณเอง 

1. รันคำสั่ง:
   ```bash
   python calibrate.py
   ```
2. มองตรงไปที่กล้อง ลืมตาปกติ 15–20 วินาที
3. เมื่อเสร็จ ให้กด Q หรือรอหมดเวลา โปรแกรมจะคำนวณ EAR เฉลี่ย
4. ค่าที่ได้จะถูกอัปเดตไปยัง config.yaml โดยอัตโนมัติ (บรรทัด ear_closed_threshold จะเปลี่ยนค่าใหม่)  

## ▶️ การรันโปรแกรมหลัก
  ```bash
   python app.py
   ```
ปุ่มลัด
Q หรือ ESC → ออกจากโปรแกรม
D → เปิด/ปิดโหมด debug (แสดงค่า EAR/MAR บน Terminal)
